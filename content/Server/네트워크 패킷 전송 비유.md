## 예시 설명

아파트 단지 내 택배를 보내는 방법으로 예시를 들어보겠다

하나의 예시로 들어보자면

같은 아파트의 사람들끼리 택배를 보내려고 한다

보내는 사람과 받는 사람을 안다면
바로 보내버리면 된다
하지만 집이 많아지면 경로를 다 외워야 한다
그래서 경비실을 중간 매개체로 사용해서

보내는 집 -> 경비실 -> 받는 집

이런 식으로 사용하면 보다 간편하게 택배를 보낼 수 있는 방법이 된다

✨하지만!

상황을 바꿔보겠다
다른 아파트끼리 택배를 보내는 방식으로 (삼성 아파트 -> 현대 아파트)

삼성 아파트 201호 -> 삼성 아파트 경비실 ->
택배 배송 센터 -> 
현대 아파트 경비실 -> 현대 아파트 101호

이런 식으로 변화가 된다

---
위에 써 놓은 비유 사물을 네트워크 방식으로 변화를 시켜보자!

비유 사물
- 같은 아파트 단지 == 네트워크
- 집 == 사용자 (pc, 모바일, 단말기) 등등
- 경비실 == 스위치
- 택배 배송 센터 == 라우터
- 택배 == 패킷
